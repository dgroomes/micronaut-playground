# basic

WORK IN PROGRESS

This sub-project is a "hello world" Micronaut application.

## Description

I scaffolded this with the Micronaut [`mn` CLI](https://docs.micronaut.io/latest/guide/index.html#buildCLI).

## Instructions

* Use Java 11
* Build and run the app:
  * `./gradlew run`
* Build and run the tests:
  * `./gradlew test`
